👀 Notice & wonder

You might notice…
  • Each row gains exactly 4 seats
  • The first row starts at 10, not 4
You might wonder…
  • How many seats in row 50 — without listing 50 rows?
  • Is there a rule that jumps straight from the row number to the seat count?

Hold on to those wonderings — this chapter answers most of them.

🔮 Discover — predict first

Look at 3, 7, 11, 15… What jumps from one term to the next? What would the 10th term be?

Each term is 4 more than the last. Starting at 3, the 10th term is 3 + 9×4 = 39. The constant jump is the key to the rule.

📖 Read the chapter

A sequence is an ordered list following a rule. Spotting the rule predicts any term without listing them all — the doorway to algebra.

A sequence is an ordered list of numbers built by a rule; each number is a term, and its position in the list is n.

  • A term-to-term rule tells you how to get the next term from the one before it, e.g. "add 4".
  • A position-to-term rule (the nth-term rule) lets you jump straight to any term — like the 100th — without listing every term before it. For example, with rule 4n + 3, term 100 = 4 × 100 + 3 = 403, with no need to list terms 1–99.
  • The common difference in a linear sequence becomes the number multiplying n; a constant is then added or subtracted to shift the rule so it matches the real first term. For example, 5, 8, 11, 14… has difference 3, so the rule starts 3n; at n = 1 that gives 3 but the term is 5, so add 2 → 3n + 2.
  • Not every pattern grows by a constant amount — square numbers and triangular numbers grow by a changing amount each step. For example, the square numbers 1, 4, 9, 16… grow by 3, 5, 7 — a gap that keeps increasing.
Two rulesterm-to-term: add the common difference
position-to-term: nth term = d × n + adjust

Going the other way — from a sequence to its rule — is the direction most questions actually ask. For 7, 11, 15, 19…: the difference is always +4, so the rule starts "4n". Testing n = 1 gives 4×1 = 4, but the real first term is 7, so add 3: the rule is 4n + 3. Check it: n = 2 → 4×2+3 = 11. ✓

When the difference itself changes, look at the second differences (the gaps between the gaps). For 2, 6, 12, 20, 30… the first differences are 4, 6, 8, 10 — each 2 more than the last — so the next gap is 12 and the next term is 42. Two families are worth knowing by name: the square numbers 1, 4, 9, 16… have nth term n², and the triangular numbers 1, 3, 6, 10… have nth term n(n + 1) ÷ 2.

🎭 Theatre seating

Row 1 has 10 seats, and each row behind has 4 more: 10, 14, 18, 22… The nth-term rule is 4n + 6, so row 20 has 4×20+6 = 86 seats — no need to count every row in between.

💰 Loan repayments

A loan balance drops the same amount each month: $2000, $1800, $1600… The term-to-term rule (subtract 200) predicts next month, but the nth-term rule predicts any month instantly.

🏗️ A growing tiled border

A tiled border around a growing pool needs 8, 12, 16, 20… tiles. Spotting the nth-term rule 4n + 4 tells you exactly how many tiles any size pool needs, without building a model first.

Why a rule like 3n + 2? If a sequence rises by 3, its rule has 3n; the '+2' shifts every term to match the real first value. Always test on term 1.
⚠ Watch outThe “add the difference” rule only gives the next term. To jump to the 50th term you need the position-to-term (nth-term) rule.
⚡ Mental maths strategyDifference × n, then adjust: a sequence rising by 4 starting at 10 is 4n + 6. Always test your rule on term 1 before trusting it.

🕵️ Mistake detective

Diagnose the misconception, then reveal the cause and intervention.

For 2, 4, 6, 8… they said the rule is × 2.
Cause

× 2 would give 2, 4, 8, 16. Check the rule against every gap, not just the first.

Diagnostic question

What rule or representation would show why this answer cannot be right?

Intervention

The gaps are all +2, so the rule is 'add 2', giving 2, 4, 6, 8, 10.
